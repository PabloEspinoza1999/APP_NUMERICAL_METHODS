Hola,

Estás usando el sistema de generación de informes de clientes.


En el menú izquierdo encontrarás las siguientes opciones:

  1. Retención de clientes con descuentos óptimos:

    Este informe ejecuta un algoritmo basado en Newton Raphson para encontrar el descuento más óptimo para un producto con el que pretendas retener clientes.

  2. Tiempo óptimo de producción para la fabricación de productos:

    Este informe permite obtener el mejor tiempo de producción en horas mediante el método de Secante, esto para optimizar el esfuerzo en tiempo del personal disponible en tu organización.

  3. Proyección soporte al cliente:

    Este informe muestra una proyección de casos abiertos y resueltos sobre solicitudes de clientes, además proyecta el tiempo promedio en un lapzo para tomar decisiones sobre el personal que atiende a los clientes de una empresa, dicho informe ejecuta un algoritmo basado en Polinomios de Newton.

  4. Proyección de satisfacción al cliente:

    Este informe calcula una proyección de satisfacción a cliente mediante un algoritmo basado en Lagrange, según el histórico de datos que se proporcione realizará una proyeccioń en caso de que algún factor como los precios de un producto, tratamiento al cliente y demás se mantuviera..

  5. Manual de usuario: Guía de usuario final

    Este apartado proporciona el acceso al manual de usuario que detalla cada una de las funcionalidades de este sistema.
